Madlibs is a game that uses a series of randomly picked words to create an out-of-context (and usually odd) story. Most people have played this game before on a piece of paper, but with the usage of variables, input functions and concatenation, we can create a simple program that will take in our words and output a story without words in it! This was one of the first projects I made using python.

The first step of building this game was finding a mad-libs example to use as a template. I just searched mine up and picked the first one I saw and set each blank as its variable, assigning its names accordingly and making sure that each differs from the other.


#The input function will take in a  the user's input and store it as a variable for me to reference
